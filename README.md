# Décroche ton oral — Backend

Backend Node.js / Express pour l’espace élève du site decroche-ton-oral.com.

## Fonctions prévues

- Connexion élève
- Sessions sécurisées
- Accès aux documents PDF protégés
- Gestion des offres
- Crédits de simulations orales
- Réservations Cal.com
- Journalisation des téléchargements

## Déploiement

Le backend tourne sur le VPS dans :

/opt/decroche-ton-oral-backend

Il est lancé avec PM2 sur le port 4100.

## Variables d'environnement

Le fichier `.env` n'est pas versionné.
Voir `.env.example`.
